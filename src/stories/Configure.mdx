    # Alto documentation
 

    This documentation enables alto assessment team to share the behaviour of components used in <a href="https://usealto.com/">Alto Assessment application</a>.  
    It's mainly used for hard to catch "edge cases".

    ### Start
    `npm run storybook`  
    Then go to [http://localhost:6006/](http://localhost:6006/)

    ### Quick tips
    - Stories must have a title describing where it belongs in our app. It makes it easier to find our components  
    So for a `ButtonGroup`, we specify `title: 'Shared/ButtonGroup'`
    - Always add the property `tags: ['autodocs'],` in order to generate automatically the "Docs" file in the documentation
